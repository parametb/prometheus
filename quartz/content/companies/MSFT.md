# MSFT — Microsoft Corporation

*บริษัทไมโครซอฟท์คอร์ปอเรชั่น*

**Sector:** Technology | **Exchange:** NASDAQ | **Updated:** 2026-04-07

Microsoft develops, licenses, and supports software, services, devices, and solutions worldwide. Key segments: Intelligent Cloud (Azure), Productivity & Business Processes (Office 365, LinkedIn), and More Personal Computing.

## Business Model

Microsoft ดำเนินธุรกิจผ่าน 3 segments หลัก: (1) Intelligent Cloud — Azure cloud platform, SQL Server, Windows Server, GitHub; (2) Productivity & Business Processes — Microsoft 365 (Office, Teams, Exchange), LinkedIn, Dynamics 365; (3) More Personal Computing — Windows OEM, Xbox, Surface, Bing Search, Advertising รายได้หลักมาจาก subscription (SaaS/PaaS) ที่ predictable และ high-margin

## Competitive Position

เบอร์ 1 ด้าน enterprise software ทั่วโลก เบอร์ 2 cloud provider (Azure ~22% vs AWS ~33%) และเป็น AI infrastructure play ที่แข็งแกร่งที่สุดในตลาดผ่าน OpenAI partnership ($13B+ ลงทุน) ทำให้ทุก Microsoft product มี AI-first positioning

### Moat Factors

- Microsoft 365 enterprise lock-in — 1.5B+ ผู้ใช้ switching cost สูงมาก เปลี่ยน productivity suite = ต้องเปลี่ยนวัฒนธรรมองค์กร
- Azure sticky infrastructure — เมื่อ migrate ขึ้น Azure แล้ว ย้ายออกยากมาก (data gravity, integration cost)
- GitHub developer ecosystem — 100M+ developers สร้างความผูกพันตั้งแต่เขียนโค้ด
- LinkedIn professional network — 1B+ members ไม่มีคู่แข่งที่ใกล้เคียงใน B2B professional data
- Enterprise trust & compliance — FedRAMP, ISO 27001, HIPAA, GDPR compliance ที่ไม่มีใครทำได้เหมือน เป็น barrier ใน regulated industries

## Investment Case

### Bull Case

- Azure กำลัง accelerate พร้อม AI tailwind ที่ demand เกิน supply — AI capacity ที่กำลังสร้างจะ unlock growth ใน H2 FY2025 Copilot monetization เพิ่งเริ่ม — $30/user/month ถ้า attach rate เพิ่มจาก 5% เป็น 20% ใน 365 base = รายได้เพิ่ม $10B+/ปี GitHub Copilot ครอง developer market ก่อน ทำให้ developer workflow ผูกกับ MSFT ecosystem ลึกขึ้น OpenAI partnership ให้ MSFT ได้ model ที่ดีที่สุด (GPT-4o, o1) ก่อนคู่แข่ง

### Bear Case

- $80B+ AI capex ใน FY2025 กดดัน FCF อย่างหนัก ถ้า AI ROI ช้ากว่าที่คาด margin compression จะยาวนานกว่าที่ตลาดคาด Azure growth deceleration — ถ้า Azure ลดจาก 30%+ เป็น 20% นัก sentiment จะเปลี่ยนเร็วมาก OpenAI ไม่ exclusive กับ MSFT — Apple, Google, Amazon ก็เข้าถึง frontier models ได้ CrowdStrike-type dependency — ถ้า Azure มี major outage กระทบ trust ใน enterprise

## Key Risks

- **AI Capex ROI Timeline** *(high)*: $80B+ AI infrastructure investment ใน FY2025 กดดัน FCF และ operating margin ในระยะสั้น ถ้า AI revenue ช้ากว่า capex ramp market จะ re-rate valuation
- **Azure Growth Deceleration** *(high)*: Consensus คาดว่า Azure จะ accelerate ในครึ่งหลัง FY2025 ถ้าไม่เป็นไปตามที่คาด sentiment จะเสีย — stock กำลัง price ใน acceleration ที่ยังไม่เกิดขึ้น
- **OpenAI Concentration Risk** *(medium)*: Microsoft ลงทุนใน OpenAI $13B+ แต่ OpenAI ไม่ exclusive — OpenAI มีดีล partnership กับ Apple, Amazon และอื่นๆ ถ้า OpenAI shift ไปหาคู่แข่ง หรือ in-house model ของ Google/Amazon ดีพอ ข้อได้เปรียบของ MSFT จะลดลง
- **Antitrust / Regulatory** *(medium)*: การเข้าซื้อ Activision Blizzard และ GitHub ทำให้ MSFT อยู่ใน regulatory scrutiny อย่างต่อเนื่อง EU กำลังตรวจสอบ Teams bundling และ cloud market dominance
